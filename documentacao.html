Documentação

Essa é a documentação onde contém todos os recursos e opções do framework

Estrutura

O Code Book segue um padrão em sua estrutura para que tude fique melhor dividio, abaixo será explicado:

Base

Encontra-se os reset's css, importação das fontes utilizadas no projeto e algumas configurações básicas no HTML.

Component

Encontra-se os componenetes, componentes são elementos que possuem sua criação personalizada para um melhor reaproveitamento de código.

Helpers

Encontra-se os arquivos para variáveis, mixin e placeholders.

Layout

Encontra-se elementos que existem apenas para modificar o visual da página, não tendo importância em sua semântica.

Object

Encontra-se todos os elementos do projeto.

States

Encontra-se classes para modificar os estados dos elementos.

Utility

Encontra-se os utilitários, utilitários são uma espécie de elementos para auxiliar os componenets e objects.
